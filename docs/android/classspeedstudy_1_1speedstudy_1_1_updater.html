Detailed Description

Make's use of Squareup's OkHttp to connect to the server. This opensource software is used to form the requests to the server as well as handle the response. Grabs all the users files from the server, and compares their date last modified with the date of the files saved locally, and updates all the files that were more recently updated on the server. Creates virtual flipcards from the data coming in from the server, and deletes flipcards that should no longer exist. Also updates a file containing information about all the users locally stored documents.
